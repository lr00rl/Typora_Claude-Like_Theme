# Release Notes - v1.2.2

## Summary

This release refines Typora code block line numbers to fix selection artifacts, improve cursor behavior, and remove delayed rendering during fast scrolling.

## Highlights

### 1. Selection and Cursor Isolation

- Fixed the issue where text selection could visually bleed into the line number area.
- Restored a dedicated left margin on `.CodeMirror-sizer` so the cursor stops at the divider instead of entering the gutter region.
- Kept line numbers non-selectable while preserving normal text selection inside code content.

### 2. Faster Line Number Rendering

- Removed dependence on Typora's lazy-rendered native CodeMirror gutter.
- Switched to a CSS counter-based line number strategy for immediate rendering during scroll.

### 3. Visual and Structural Cleanup

- Added a subtle vertical divider between line numbers and code content.
- Improved layering so line numbers remain visible above selection backgrounds.
- Tuned width, spacing, opacity, and alignment for more stable presentation.

### 4. Theme Consistency

- Applied the same behavior to both `claude-like.css` and `claude-like-dark.css`.
